Pear Wagon Game

This is a game where the user stears a wagon to get points by catching green pears and avoid bombs and rotten pears. This was a collaboration with Anton, Jonas, Petter, Yohan and Andreas. The prototype was created with Figma to get an idea of how the game would look like. Object oriented programming was our working method. To build this we used P5.js and Typescript. We used to Git to wok together.
